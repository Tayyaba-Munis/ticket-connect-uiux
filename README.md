# 🎫 Ticket Connect – UI/UX Design Project

A UI/UX design and prototype for a smart ticket booking system focused on simplicity, speed, and better user experience.

## 👩‍💻 Role
UI/UX Designer & Prototyping

- Designed full user flow and wireframes in Figma  
- Created high-fidelity UI screens  
- Built interactive prototype  
- Focused on usability and mobile-first design  

## 🎯 Goal
- Simplify ticket booking process  
- Improve navigation and usability  
- Optimize seat selection & checkout flow
  
## 🖼️ Screens
- Home  
- Events  
- Event Details  
- Seat Selection  
- Payment  
- Confirmation
